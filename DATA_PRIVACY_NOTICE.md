# NCKUSH Sleep Apnea Dataset - Data Privacy

## ⚠️ CONFIDENTIAL - DO NOT DISTRIBUTE

This folder contains **SENSITIVE PATIENT DATA** from National Cheng Kung University Hospital Sleep Center (NCKUHSC).

### Data Protection Requirements

1. **🔒 Access Control**
   - Data access restricted to authorized researchers only
   - IRB approval required for any data usage
   - This data is NOT included in the GitHub repository

2. **🚫 Prohibited Actions**
   - DO NOT upload to any public repository (GitHub, GitLab, etc.)
   - DO NOT share via email, cloud storage, or messaging
   - DO NOT copy to personal devices without authorization
   - DO NOT include in presentations or publications without de-identification

3. **📋 Usage Guidelines**
   - Use only for approved research purposes
   - Process data on secure lab computers only
   - Delete all copies after project completion
   - Report any data breach immediately

### Files That Must NEVER Be Uploaded

```
❌ *.edf, *.EDF          # Raw ECG recordings
❌ *.xml, *.XML          # Sleep study annotations
❌ *.npy                 # Preprocessed ECG segments
❌ Subject metadata       # Any patient identifiable information
❌ Spectrograms with IDs # If containing subject identifiers
```

### What CAN Be Shared

```
✅ Aggregated statistics (no individual data)
✅ Model architectures and code
✅ Averaged performance metrics
✅ De-identified visualization examples
✅ Research methodology and documentation
```
